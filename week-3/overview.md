# Week 3 — Advanced Patterns & Real Use Cases

## Weekly Theme
*"Build real things. Games, auctions, identity, confidential tokens."*

---

## Learning Objectives

1. Build an encrypted game using branch-free feedback logic
2. Implement a sealed-bid auction with encrypted bids and public reveal
3. Design an identity registry with encrypted attributes and compliance checks
4. Use OpenZeppelin Confidential's ERC7984 standard
5. Implement cross-contract FHE flows using `allowTransient`
