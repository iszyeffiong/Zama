# ERC7984 Standard

**The confidential token standard for FHEVM**, analogous to ERC20 but with encrypted balances and transfer amounts.

## Differences from ERC20

| Feature | ERC20 | ERC7984 |
|---|---|---|
| Balances | Public `uint256` | Encrypted `euint64` |
| Transfer amounts | Public | Encrypted |
| `balanceOf` return | Plaintext | Encrypted handle |
| Approvals | Public amount | Encrypted amount |
| Events | Amounts visible | Amounts hidden |

## Installation

```bash
npm install @openzeppelin/confidential-contracts
```

```solidity
import "@openzeppelin/confidential-contracts/token/ERC7984/ERC7984.sol";
```

## Key Functions

```solidity
// Mint confidential tokens
function _mint(address to, euint64 amount) internal

// Transfer — amount stays encrypted
function transfer(address to, externalEuint64 amount, bytes calldata proof) external

// View encrypted balance (handle — decrypt off-chain)
function balanceOf(address account) external view returns (euint64)

// Approve encrypted spending allowance
function approve(address spender, externalEuint64 amount, bytes calldata proof) external
```

## Variants

| Contract | Description |
|---|---|
| `ERC7984` | Base confidential token |
| `ERC7984Restricted` | With transfer restrictions (KYC hooks) |
| `ERC7984ObserverAccess` | Opt-in audit/compliance observers |
| `ERC7984ERC20Wrapper` | Wrap/unwrap public ERC20 ↔ confidential ERC7984 |
