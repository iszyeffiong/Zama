# Week 1 — FHE Foundations & First Contracts

## Weekly Theme
*"Understand the model. Write your first encrypted contract. See the magic."*

---

## Learning Objectives

By the end of Week 1, participants will be able to:

1. Explain what FHEVM is and why it matters for Web3 privacy
2. Understand the handle model and why encrypted types behave differently from plaintext types
3. Set up a local FHEVM development environment from scratch
4. Write, test, and deploy a contract that stores and retrieves an encrypted value
5. Perform basic FHE arithmetic (add, sub, compare) on encrypted values
6. Implement both user decryption and public decryption patterns
